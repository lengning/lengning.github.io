EBSeq

EBSeq is an R package for identifying differentially expressed genes and isoforms using RNA-seq data. EBSeq provides improved power and well-controlled FDR in isoform analysis by accounting for varying uncertainty across different isoform groups. It accommodates experiments with two or more conditions. EBSeq graphical user interface and Galaxy wrapper are also available on EBSeq page.

EBSeqHMM

EBSeqHMM is an R package for detecting expression changes in RNA-seq experiments with ordered conditions (e.g. time or spatial course). EBSeqHMM incorporates a hidden markov component to accommodate dependence between adjacent time points (positions).
